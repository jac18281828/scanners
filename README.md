# Scanners

A native macOS (arm64) document/image scanning app built on SANE, targeting the HP
ScanJet 4570c and other SANE-supported scanners. Scan to searchable PDF or image, no
GNOME Document Scanner required.

[![CI](https://github.com/jac18281828/scanners/actions/workflows/ci.yml/badge.svg)](https://github.com/jac18281828/scanners/actions/workflows/ci.yml)

## Getting Started

The app is not yet packaged for distribution — binary install (download the `.app`,
drag to `Applications`) lands in a later phase. For now this is a SwiftPM package under
active development; see `DESIGN.md` for architecture and `STATE.md` for current phase
status.
